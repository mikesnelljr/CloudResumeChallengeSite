Establishing the foundations - Cloud Resume Challenge


I would like to start off by saying that I am by no means a professional web developer. It has been quite a long time since I have played with web development. With that in mind, I decided to start off with a template, reverse engineer it and then customize. I got the base for this site from html5up.net and this particular template is called "hyperspace". I spent more time on this part than I probably should have but what can I say? Creative works are an outlet of mine. So, why not give it the ol' razzle dazzle while I duct tape some things together?

This site was built locally on my machine in vscode and pushed to GitHub. And from there GitHub syncs with AWS to populate the files in S3 to feed cloudfront.

Link to template: https://html5up.net/hyperspace
